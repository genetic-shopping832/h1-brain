## 🧠 What is h1-brain?

h1-brain is a simple tool that links AI assistants with HackerOne. It helps automate tasks in bug bounty hunting. You do not need to know how to code to use it. The server runs on your Windows computer and talks directly to HackerOne, which is a platform for finding and reporting security issues.

---

## 🎯 Who is this for?

This application is for anyone interested in bug bounty hunting or security testing without needing technical skills. It can help you:

- Automate hacker tasks
- Manage bug bounty jobs easily
- Connect AI with real bug hunting platforms

## 💻 System Requirements

To run h1-brain on your Windows PC, your system should meet these minimum requirements:

- Windows 10 or later (64-bit)
- At least 4 GB of RAM
- 500 MB free disk space
- Internet connection to communicate with HackerOne

## 🌀 How h1-brain works

h1-brain runs as a server on your Windows system. It acts as a bridge between an AI assistant and HackerOne. The AI helps find security vulnerabilities, while h1-brain manages communication and actions through HackerOne’s system.

No special setup for AI or HackerOne’s API keys is needed here, as the app guides you through the process step-by-step.
